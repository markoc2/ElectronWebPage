    Cargando...

    Open Visual Studio 2019

    1. On the start window, choose Crear un nuevo proyecto.

    2. On the Crear un nuevo proyecto window, enter or type Windows Forms in the search box. Next, choose Desktop from the Tipo de proyecto list.

      After you apply the Tipo de proyecto filter, seleccionamos Windows Forms App (.NET Framework) template for either C# or Visual Basic, and then choose Next.

      Note

      If you don't see the Windows Forms App (.NET Framework) template, you can install it from the Crear un nuevo proyecto window. In the Not finding what you're looking for? message, seleccionamos Install more tools and features link.

      Next, in the Visual Studio Installer, seleccionamos .NET desktop development workload.

      After that, seleccionamos Modify button in the Visual Studio Installer. You might be prompted to save your work; if so, do so. Next, choose Continue to install the workload.

    3. In the Configure your new project window, type or enter PictureViewer in the Project name box. Then, choose Create.
