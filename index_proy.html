Abrir Visual Studio

  1. En la ventana de inicio, Seleccione Crear un nuevo proyecto.

  2. En la ventana Crear un nuevo proyecto, en la seccion de busqueda introduzca Windows Forms. Despues, Seleccione Desktop en la lista de Tipo de proyecto.

    Despues de aplicar el filtro de Tipo de proyecto, seleccionamos Windows Forms App (.NET Framework) plantilla para C # o Visual Basic, y luego Seleccione Siguiente.

    Nota

    Si no ve la plantilla de la aplicación Windows Forms App (.NET Framework) , puede instalarla desde la ventana Crear un nuevo proyecto. En el ¿No encuentras lo que estás buscando? mensaje, seleccionamos el enlace Instalar más herramientas y características.

    Despues, en el instalador de visual Visual Studio, seleccionamos .NET desktop development en carga de trabajo.

    Luego de eso, seleccionamos el botón Modificar en el Visual Studio Installer. Es posible que se le solicite que guarde su trabajo; Si es así, hazlo. A continuación, seleccione Continuar para instalar la carga de trabajo.

  3. En la ventana de Configuración de nuevo proyecto, introduzca Proyectos_Graficos como Nombre del Proyecto en le cuadro de texto. Luego, Seleccione Crear.
